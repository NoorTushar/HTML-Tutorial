# HTML Tutorials by Noor Tushar - Module 04

### Semantic HTML

```html
<body>
  <header>
    <nav>
      <ul>
        <li><a></a></li>
        <li><a></a></li>
        <li><a></a></li>
      </ul>
    </nav>
  </header>
  <main>
    <article>
      <header>
        <!-- Contains title and also time usually -->
        <time datetime=""></time>
      </header>
      <figure>
        <img src="" />
        <figcaption></figcaption>
      </figure>
      <section>
        <!-- first section -->
      </section>
      <section>
        <!-- second section -->
      </section>
      <section>
        <!-- third section -->
      </section>
      <footer>
        <!-- might contain tag links -->
      </footer>
    </article>
  </main>
  <footer>
    <!-- Footer Section -->
  </footer>
</body>
```
